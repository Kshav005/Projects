Inspired by slither.io, the game features a smooth gameplay of 2D elements, all created using Pygame. The menu, songs, main game, everything is created by my own, 100% original. QR Code Generator

Tkinter, QR code module and Pillow were used in order to complete the project. You can even use an API to convert a link but I got the module, so I did it using it. Pillow will change the format of the QR code into a readable image format. Tkinter was only used for GUI, getting the link from the user and thus opening up a QR code photo upon confirming.

Internet Speed Test

One more nice application which can measure your upload/download speed according to the server you choose. It can be either closest server or the best server in order to check speed. Also, a speedtest module is used in order to perform such actions.

Data Storage

This program uses JSON to store and retrieve data. You can even edit your information and it works on the basis of a passcode. Basically it's an employee data storage. You can even add to show the whole data and delete some employee data. I only created the upload and edit data because it's my first project related to permanent storage of data. You can add as many features as you want. Refer to the source code.

Bill Invoice

Another storage system but it works differently. It is created for a shopkeeper who delivers some items and if he/she presses save, then the computer creates an invoice folder along with the data, then saves a notepad file which contains the invoice no., customer name, items they ordered and the total amount. The price is pre-defined in a json file.
